# my-stock-team

**종목 이름만 말하면, AI 애널리스트 5명이 협업해서 주식 리서치 리포트를 만들어 주는 도구입니다.**

예를 들어 "삼성전자 분석해줘"라고 입력하면 재무 분석, 주가 흐름 분석, 뉴스 분석을 거쳐
검증까지 마친 리포트 문서가 자동으로 만들어집니다. PPT 파일로도 변환할 수 있습니다.

> ⚠️ 무료 공개 데이터를 활용한 **학습용 도구**입니다. 투자 권유가 아니며,
> 모든 투자 판단과 책임은 본인에게 있습니다.

---

## 1. 시작 전에 필요한 것

| 필요한 것 | 설명 |
|---|---|
| **Claude Code** | 이 도구가 돌아가는 AI 프로그램입니다. [설치 안내](https://claude.com/claude-code)를 따라 설치하세요. |
| **Python(파이썬)** | 차트와 PPT를 만들 때 필요한 프로그램입니다. [python.org](https://www.python.org/downloads/)에서 무료로 설치할 수 있습니다. |
| **DART API 키** | 금융감독원의 기업 공시 데이터를 가져오기 위한 "입장권" 같은 것입니다. 무료이며, 발급 방법은 아래 3번에서 설명합니다. |

## 2. 설치하기

Claude Code를 실행한 다음, 아래 두 줄을 **한 줄씩** 입력하면 설치 끝입니다.
(복사해서 붙여넣고 Enter를 누르면 됩니다.)

```
/plugin marketplace add DAJUNG03/my-stock-team
/plugin install my-stock-team@my-stock-team
```

그리고 Claude Code에 아래 문장을 입력해서 필요한 파이썬 패키지 설치를 부탁하세요.

```
pip install python-pptx matplotlib finance-datareader pykrx pandas python-dotenv 실행해줘
```

## 3. DART API 키 발급받기 (한 번만 하면 됩니다)

이 키는 **각자 본인 것을 발급받아야 합니다.** 발급은 무료이고 5분이면 됩니다.

1. [opendart.fss.or.kr](https://opendart.fss.or.kr) 접속 → 회원가입
2. 로그인 후 **[인증키 신청/관리]** 메뉴에서 인증키 신청
3. 이메일로 받은 인증키(영문+숫자 40자)를 복사해 둡니다
4. 분석 작업을 할 폴더 안에 `.env` 라는 이름의 파일을 만들고, 안에 아래 한 줄을 적습니다

   ```
   DART_KEY=여기에_발급받은_키_붙여넣기
   ```

   💡 파일 만드는 게 어렵다면, Claude Code에 이렇게 부탁해도 됩니다:
   **".env 파일 만들어줘. DART_KEY는 ○○○이야"**

> 🔒 이 키는 비밀번호처럼 다뤄 주세요. 다른 사람에게 공유하거나 인터넷에 올리면 안 됩니다.

## 4. 사용하기

Claude Code에 평소 말하듯이 입력하면 됩니다.

```
삼성전자 분석해줘
```

그러면 자동으로 이런 일이 일어납니다:

1. **재무 애널리스트**가 회사의 실적·재무 상태를 분석하고
2. **차트 애널리스트**가 주가 흐름과 거래량을 분석하고
3. **뉴스 애널리스트**가 최근 뉴스와 시장 분위기를 정리한 뒤
4. **리스크 매니저**가 위 결과를 종합해 주의할 점을 짚고
5. **검증 담당자**가 리포트에 틀린 수치가 없는지 검사합니다

완성된 리포트는 `reports` 폴더 안에 `삼성전자.md` 같은 문서 파일로 저장됩니다.

PPT 파일로 만들고 싶다면 이어서 이렇게 입력하세요:

```
/report-pptx 삼성전자
```

## 5. 자주 묻는 질문

**Q. 리포트의 숫자는 믿을 수 있나요?**
모든 수치에 출처와 기준일이 함께 표기되고, 확인할 수 없는 정보는 "확인 불가"로
정직하게 적도록 되어 있습니다. 그래도 중요한 결정 전에는 원본 공시를 직접 확인하세요.

**Q. "사세요/파세요" 같은 추천도 해주나요?**
아니요. 이 도구는 판단에 필요한 **근거 정리까지만** 해 줍니다. 매수·매도·목표가 같은
투자 행동 추천은 하지 않도록 설계되어 있습니다. 최종 판단은 사람의 몫입니다.

**Q. 돈이 드나요?**
이 도구 자체와 DART 키는 무료입니다. 단, Claude Code 사용료는 본인 요금제를 따릅니다.

**Q. 어떤 종목이든 되나요?**
한국 상장 기업(코스피·코스닥)을 기준으로 만들어졌습니다.
